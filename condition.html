🔄 CÂU ĐIỀU KIỆN

Conditional Sentences - Tổng hợp đầy đủ và chi tiết nhất

📋 TỔNG QUAN VỀ CÂU ĐIỀU KIỆN

🎯 Định nghĩa

Câu điều kiện là câu diễn tả một hành động có thể xảy ra khi có điều kiện nào đó. Câu điều kiện gồm 2 mệnh đề:

  • Mệnh đề điều kiện (If clause): Bắt đầu bằng "If"
  • Mệnh đề chính (Main clause): Diễn tả kết quả
Loại Điều kiện Cấu trúc If clause Cấu trúc Main clause Ví dụ
Loại 0 Chân lý, sự thật hiển nhiên If + S + V(s/es) S + V(s/es) If you heat water, it boils.
Loại 1 Có thể xảy ra ở hiện tại/tương lai If + S + V(s/es) S + will + V If it rains, I will stay home.
Loại 2 Không có thật ở hiện tại If + S + V2/ed S + would + V If I were rich, I would travel.
Loại 3 Không có thật trong quá khứ If + S + had + V3/ed S + would have + V3/ed If I had studied, I would have passed.
📝 Lưu ý quan trọng:
  • Có thể đảo vị trí 2 mệnh đề: Main clause + if clause
  • Khi mệnh đề if đứng trước, cần có dấu phẩy ngăn cách
  • Unless = If not (trừ khi)
  • Có thể dùng were cho tất cả ngôi trong câu điều kiện loại 2

🔢 CÂU ĐIỀU KIỆN LOẠI 0

🔢
Cấu trúc cơ bản
If clause: If + S + V(s/es)
Main clause: S + V(s/es)

Có thể thay If bằng: When, Whenever
Ví dụ:
• If you heat ice, it melts.
• When you mix red and blue, you get purple.
• If you don't water plants, they die.
💡
Cách dùng chi tiết
🎯 Sử dụng khi:
• Diễn tả chân lý, sự thật hiển nhiên
• Quy luật tự nhiên, khoa học
• Thói quen, hành động lặp đi lặp lại
• Hướng dẫn, chỉ dẫn
Ví dụ thực tế:
• If you press this button, the machine starts.
• If it's sunny, we usually go to the beach.
• If you study hard, you get good grades.
⚠️ Chú ý: Câu điều kiện loại 0 không dùng "will" trong cả hai mệnh đề vì diễn tả sự thật chắc chắn, không phải dự đoán.

1️⃣ CÂU ĐIỀU KIỆN LOẠI 1

1️⃣
Cấu trúc cơ bản
If clause: If + S + V(s/es)
Main clause: S + will + V

Biến thể:
• can/may/must/should + V
• Câu mệnh lệnh
Ví dụ:
• If it rains tomorrow, I will stay home.
• If you finish your homework, you can watch TV.
• If you see John, tell him to call me.
🎯
Cách dùng chi tiết
💡 Sử dụng khi:
• Điều kiện có thể xảy ra ở hiện tại/tương lai
• Dự đoán có căn cứ
• Lời đe dọa, cảnh báo
• Lời hứa, đề nghị
Các tình huống thực tế:
• If you don't hurry, you will miss the bus.
• If you help me, I will help you too.
• If she studies hard, she will pass the exam.
⚡
Các biến thể đặc biệt
1. Unless (trừ khi):
Unless + S + V = If + S + don't/doesn't + V

2. Provided that/Providing that:
Provided that + S + V

3. As long as/So long as:
As long as + S + V
Ví dụ:
• Unless you study, you will fail.
• Provided that you pay on time, there's no problem.
• As long as you're happy, I'm happy.
📝
Các thì khác trong If clause
Present Continuous:
If + S + am/is/are + V-ing

Present Perfect:
If + S + have/has + V3/ed
Ví dụ:
• If you are coming, please let me know.
• If you have finished your work, you can go home.

2️⃣ CÂU ĐIỀU KIỆN LOẠI 2

2️⃣
Cấu trúc cơ bản
If clause: If + S + V2/ed
Main clause: S + would + V

Đặc biệt với "be":
If + S + were (tất cả ngôi)
Ví dụ:
• If I were rich, I would buy a big house.
• If she studied harder, she would pass the exam.
• If I had time, I would learn guitar.
🎭
Cách dùng chi tiết
🎯 Sử dụng khi:
• Điều kiện không có thật ở hiện tại
• Tình huống tưởng tượng
• Lời khuyên một cách lịch sự
• Điều ước không thể thực hiện
Tình huống thực tế:
• If I were you, I would apologize.
• If we had more money, we would travel more.
• If she were here, she would help us.
🔄
Các biến thể với modal verbs
Would/Could/Might + V:
• would = chắc chắn xảy ra
• could = có thể, có khả năng
• might = có thể (ít chắc chắn hơn)
So sánh:
• If I won the lottery, I would quit my job. (chắc chắn)
• If I won the lottery, I could quit my job. (có thể)
• If I won the lottery, I might quit my job. (không chắc)
💫
Cấu trúc đặc biệt
1. If I were you:
If I were you, I would...

2. Wish (ước):
S + wish + S + V2/ed
Ví dụ:
• If I were you, I would study medicine.
• I wish I had a bigger house.
• She wishes she could speak French.

3️⃣ CÂU ĐIỀU KIỆN LOẠI 3

3️⃣
Cấu trúc cơ bản
If clause: If + S + had + V3/ed
Main clause: S + would have + V3/ed
Ví dụ:
• If I had studied harder, I would have passed the exam.
• If she had known, she would have come.
• If they had left earlier, they wouldn't have missed the train.
⏰
Cách dùng chi tiết
🎯 Sử dụng khi:
• Diễn tả điều kiện không có thật trong quá khứ
• Hối tiếc về việc đã không làm trong quá khứ
• Phê bình, chỉ trích về hành động trong quá khứ
• Đưa ra lời giải thích cho sự việc đã xảy ra
Tình huống thực tế:
• If I had known you were sick, I would have visited you.
• If he had driven carefully, he wouldn't have had an accident.
• If we had taken a map, we wouldn't have gotten lost.
🔄
Các biến thể với modal verbs
Would/Could/Might have + V3:
• would have = chắc chắn xảy ra
• could have = có thể, có khả năng
• might have = có thể (ít chắc chắn hơn)
So sánh:
• If I had known, I would have helped. (chắc chắn)
• If I had known, I could have helped. (có thể)
• If I had known, I might have helped. (không chắc)
💭
Cấu trúc đặc biệt
1. Wish (ước trong quá khứ):
S + wish + S + had + V3/ed

2. If only (Giá mà):
If only + S + had + V3/ed
Ví dụ:
• I wish I had studied harder for the test.
• If only I had listened to your advice.
• She wishes she had taken that job offer.

🔀 CÂU ĐIỀU KIỆN HỖN HỢP

🔀
Loại hỗn hợp 1: Quá khứ - Hiện tại
If clause: If + S + had + V3/ed
Main clause: S + would + V
Ví dụ:
• If I had studied medicine, I would be a doctor now.
• If you had saved money, you could buy a house today.
• If she had learned to drive, she wouldn't need to take the bus.
🔄
Loại hỗn hợp 2: Hiện tại - Quá khứ
If clause: If + S + V2/ed
Main clause: S + would have + V3/ed
Ví dụ:
• If I were smarter, I would have gone to university.
• If she liked children, she would have become a teacher.
• If he had more time, he would have finished the project.
🎯
Cách dùng chi tiết
💡 Sử dụng khi:
• Kết quả ở hiện tại của điều kiện trong quá khứ
• Kết quả trong quá khứ của điều kiện ở hiện tại
• Diễn tả sự hối tiếc với tình huống hiện tại
• Giải thích tình huống hiện tại bằng điều kiện quá khứ
Tình huống thực tế:
• If I had taken that job, I would be living in Paris now.
• If I were more confident, I would have asked for a promotion.
• If you weren't so stubborn, we wouldn't have argued yesterday.
⚠️
Lưu ý quan trọng
Chú ý khi sử dụng:

• Câu điều kiện hỗn hợp cần đảm bảo logic về thời gian
• Mệnh đề If thường diễn tả điều kiện trái với quá khứ
• Mệnh đề chính diễn tả kết quả trái với hiện tại
• Cần chú ý đến ngữ cảnh để sử dụng cho phù hợp

⭐ CÂU ĐIỀU KIỆN ĐẶC BIỆT

🔁
Đảo ngữ trong câu điều kiện
Loại 1: Should + S + V, S + will + V

Loại 2: Were + S + to V, S + would + V

Loại 3: Had + S + V3/ed, S + would have + V3/ed
Ví dụ:
• Should you see Tom, tell him to call me.
• Were I you, I would accept the offer.
• Had I known, I would have helped.
🔀
Unless, Provided that, As long as
Unless = If not: Unless + S + V

Provided that: Provided that + S + V

As long as: As long as + S + V
Ví dụ:
• Unless you study, you'll fail. = If you don't study, you'll fail.
• I'll help provided that you try.
• You can go as long as you finish your homework.
💭
Suppose/Supposing, What if
Suppose/Supposing: Suppose/Supposing + S + V, ...

What if: What if + S + V, ...
Ví dụ:
• Suppose you won the lottery, what would you do?
• Supposing it rains, what will we do?
• What if she doesn't come, should we start without her?
🔄
But for, Without, Otherwise
But for: But for + N, S + would have + V3/ed

Without: Without + N, S + would + V

Otherwise: S + V, otherwise + S + would + V
Ví dụ:
• But for your help, I would have failed.
• Without water, we would die.
• I studied hard, otherwise I would have failed.

✍️ LUYỆN TẬP CÂU ĐIỀU KIỆN

Câu 1: Hoàn thành câu sau: If I _____ (have) enough money, I would buy a new car.

Câu 2: Hoàn thành câu sau: If she _____ (study) harder, she would have passed the exam.

Câu 3: Hoàn thành câu sau: If it rains tomorrow, we _____ (cancel) the picnic.

Câu 4: Viết lại câu sử dụng unless: You will fail if you don't study.

Câu 5: Chuyển sang câu điều kiện: I didn't know you were coming, so I didn't wait for you.

📝 Mẹo làm bài:
  • Xác định loại câu điều kiện dựa trên ngữ cảnh
  • Nhớ công thức của từng loại điều kiện
  • Chú ý đến dấu hiệu thời gian trong câu
  • Với unless, nhớ rằng unless = if not
  • Kiểm tra lại thì của động trước khi kết luận
